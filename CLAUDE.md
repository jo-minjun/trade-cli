# trade-cli

## Language

All code, comments, commit messages, and documentation must be written in English.

## Changelog

- Maintain `CHANGELOG.md` following [Keep a Changelog](https://keepachangelog.com/en/1.1.0/) format.
- Add all notable changes under the `[Unreleased]` section as work progresses.
- When releasing, move `[Unreleased]` entries to a new versioned section with the release date.
- Group changes by: Added, Changed, Deprecated, Removed, Fixed, Security.
